# Project Name: CodeCraftHub LMS

## Repository Name
CodeCraftHub_LMS

## 📌 Project Description
CodeCraftHub LMS is a Learning Management System designed to manage courses, users, and learning activities efficiently. It provides features for students and administrators to interact, track progress, and manage educational content.

## 🚀 Features
- User Registration and Login
- Course Management
- Student Dashboard
- Admin Panel
- Content Upload and Access
- Progress Tracking

## 🛠️ Technologies Used
- Python
- Django
- HTML, CSS, JavaScript
- SQLite Database

## 📂 Project Structure
- `frontend/` – Contains static files (HTML, CSS, JS)
- `backend/` – Contains Django server code
- `templates/` – HTML templates
- `static/` – CSS, JS, images
